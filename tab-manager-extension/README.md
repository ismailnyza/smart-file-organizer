# Tab Manager Pro - Browser Extension

Organize and manage browser tabs efficiently. Group by domain, close duplicates, save/load sessions.

## Features (Free Version)
- 🌐 **Group tabs by domain** - Automatically group tabs from same website
- 🗑️ **Close duplicate tabs** - Remove tabs with same URL
- 💾 **Save sessions** - Save all open tabs for later
- 📂 **Load sessions** - Restore saved sessions
- 📊 **Tab count** - See how many tabs are open

## Premium Features ($5)

- 🕐 **Session scheduling** - Auto-save sessions on schedule
- 📈 **Tab analytics** - See tab usage statistics
- 🚫 **No watermark** - Clean interface
- 🎯 **Priority support** - Faster help

## Installation

### Method 1: Sideload (Free)
1. Download `tab-manager-pro.zip`
2. Extract to a folder
3. Open Chrome/Edge: `chrome://extensions/`
4. Enable "Developer mode" (top right)
5. Click "Load unpacked"
6. Select the extracted folder
7. Extension installed!
