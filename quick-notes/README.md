# Quick Notes Pro

A privacy-first notes Progressive Web App (PWA) that works offline.

## Features
- **Works offline** - Notes saved locally in your browser
- **Progressive Web App** - Install on phone/desktop
- **Privacy focused** - No tracking, no ads
- **Simple interface** - Clean and easy to use
- **Export/Import** - Backup your notes
